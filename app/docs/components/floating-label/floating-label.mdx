## Disabled state
Apply the disabled attribute to the input fields to disallow the user from changing the content.

<CodePreview title="Floating Label" className="grid grid-flow-col justify-stretch space-x-4">
    <FloatingLabel buttonStyle="filled" label="Label" disabled={true} />
    <FloatingLabel buttonStyle="outlined" label="Label" disabled={true}/>
    <FloatingLabel buttonStyle="standard" label="Label" disabled={true}/>
</CodePreview>
