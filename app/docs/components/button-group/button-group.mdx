The accordion component can be used to provide information to your users such as success or error messages, but also highlighted information complementing the normal flow of paragraphs and headers on a page. Flowbite also includes dismissable accordions which can be hidden by the users by clicking on the close icon.

## Set up.

Import Accordion and set variables in the script tag.

```react example
<script>
  import { Avatar } from 'flowbite-svelte';
</script>
```
